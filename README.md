# Penyelesaian Permainan Queens Linkedin [Tugas Kecil 1 IF2211 Strategi Algoritma]

![Alt text](src/assets/image.png) <br>
Program solver problem game Queens LinkedIn dengan algoritma brute force yang ditulis dalam bahasa python. Program menggunakan library pillow untuk menggambar dan tkinter untuk GUI.

# Requirements
Pastikan ini semua sudah terinstall pada perangkat (Program ini diuji pada WSL Archlinux di Windows 11):
- Python 
- NumPy
- Pillow
- Tkinter

# Cara menjalankan
1. Jalankan script main.py dari command line atau vscode
2. Klik tombol Start Solving dan pilih file .txt yang sesuai
3. Tunggu program selesai, jika ditemukan solusi gambar nya akan disimpan di folder output dengan bentuk .png
4. Klik tombol Coba Lagi apabila ingin menguji dengan kasus lain
